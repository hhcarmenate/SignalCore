# Entities

## Core entities

### Symbol
Represents a tradable market symbol such as `AAPL`, `QQQ`, or `SPY`.

### Watchlist
Represents a user-defined list of symbols to scan.

### Watchlist Item
Represents a symbol membership inside a watchlist.

### Bot
Represents a scanning strategy implementation.

### Bot Run
Represents a single execution instance of a bot.

### Signal
Represents an opportunity detected by a bot.

### Notification
Represents a UI-facing notification generated by the system.

### Signal Outcome
Represents post-signal performance tracking for analytics and bot comparison.
